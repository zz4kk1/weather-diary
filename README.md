# Weather Diary (Дневник погоды)

**Автор:** Габдулхаков Динар

## Краткое описание программы
Weather Diary — это десктопное приложение с графическим интерфейсом (GUI), разработанное на Python (Tkinter). Программа позволяет вести дневник наблюдений за погодой: добавлять записи о температуре, наличии осадков и текстовом описании погоды. Все данные сохраняются локально в формате JSON. Также реализована функция удобной фильтрации записей по конкретной дате или минимальной температуре.

## Как запустить приложение
1. Убедитесь, что на вашем компьютере установлен Python (версии 3.6 или выше).
2. Склонируйте репозиторий:
   `git clone https://github.com/zz4kk1/weather-diary.git`
3. Перейдите в папку проекта:
   `cd weather_diary`
4. Запустите файл приложения:
   `python weather_diary.py` (или `python3 weather_diary.py` на Mac/Linux)

## Примеры использования
1. **Добавление записи**: Введите дату `15.05.2024`, температуру `18.5`, описание `Солнечно, легкий ветер` и выберите осадки `Нет`. Нажмите «Добавить запись».
2. **Валидация**: Попробуйте ввести текст вместо числа в поле температуры — программа выдаст предупреждение.
3. **Фильтрация**: В блоке "Фильтры" введите `10` в поле "Температура выше (°C)" и нажмите "Применить". В таблице останутся только те дни, где было теплее 10 градусов.
